# 💰 AttentionPay — Get Paid to Watch Ads

**Your attention is worth money. Stop giving it away for free.**

Watch ads → Earn USDC on Base chain → Withdraw instantly. Zero fees.

## Features

- ♾️ **Unlimited Ads** — No caps, watch as many as you want
- ⚡ **Instant Withdrawal** — USDC on Base chain, zero gas fees
- 📱 **All Devices** — Mobile, tablet, desktop
- 🌍 **Global** — Available worldwide, no restrictions
- 📊 **Detailed Analytics** — Per-ad earning breakdowns
- 🔐 **Decentralized** — Smart contract powered payouts

## Ad Types

| Type | Earning |
|---|---|
| 🎬 Video Ads | $0.01-0.05 per view |
| 📰 Banner Ads | $0.001-0.005 per view |
| 📝 Surveys | $0.20-2.00 per survey |
| 📱 App Installs | $0.50-5.00 per install |
| 🔗 Crypto Signups | $2.00-10.00 per signup |

## Quick Start

```bash
# Install dependencies
cd server && npm install

# Start server
npm start

# Open http://localhost:3000
```

## Tech Stack

- Frontend: HTML/CSS/JS (3D Glass Morphism UI)
- Backend: Node.js + Express
- Database: lowdb (JSON)
- Payments: USDC (Base Chain)

## Revenue Model

1. Google AdSense (display/video ads)
2. Crypto Affiliate Programs (Binance, Coinbase, etc.)
3. Offer Walls (OfferToro, AdGate)
4. Sponsored Tasks
